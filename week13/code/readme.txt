kill -l       可以查看所有的信号

./1    执行1程序后，在另一个终端通过ps -ef来查看它的PID，或者直接看它打印出来的PID，然后通过kill -SIGNAL PID来给这个进程发送信号，其中SIGNAL为信号类型，PID为该进程的PID，1程序中接收了1/2/3这3种信号即SIGHUP/SIGINT/SIGQUIT。最后记得用kill -9 PID来结束进程  这个可以不开另一个终端，只要用./1&来让1程序后台运行

./2    执行2程序后，通过Ctrl+C就可以让这个程序接收并处理这个信号，这里是打印出一个字符串.这个程序使用了sigaction结构体的sa_handler来处理，它的处理过程和上面1程序用的signal一样，都是只能传一个值，即信号值  最后记得用kill -9 PID来结束进程
